# Distance Measurement System using Arduino UNO

## Description
This project measures the distance of an object using the HC-SR04 ultrasonic sensor and displays the result on an LCD/OLED display.

## Components Used
- Arduino UNO
- HC-SR04 Ultrasonic Sensor
- LCD/OLED Display
- Jumper Wires
- Breadboard

## Software Used
- Arduino IDE
- Wokwi Simulator
